# Идеи для GCC 2026 Mini App

## Интерактив на мероприятии
- **Живые опросы** — спикер задаёт вопрос, участники голосуют в приложении, результаты на экране в реальном времени
- **Q&A к спикеру** — участники пишут вопросы в приложении, организатор видит и выбирает лучшие
- **Фотостена** — участники загружают фото с хэштегом, общая галерея события
- **Квест по выставке** — "посетите 5 стендов, получите приз", прогресс в приложении

## Нетворкинг
- **Цифровая визитка** — QR-код внутри приложения, сканируют → контакт сохранён
- **Мэтчинг** — "кто из участников вам интересен?" система рекомендаций по профилю
- **Запрос на встречу** — кнопка "хочу встретиться" → уведомление другому участнику

## Демонстрация клиентам (WOW-фактор)
- **AI-консьерж** — отвечает на вопросы голосом и текстом ⭐ самое сильное
- **Push-уведомления** — "через 10 минут начинается сессия X"
- **Персональная программа** — каждый участник отмечает интересные сессии, свой расписание
- **Оценки и отзывы** — рейтинг спикеров и сессий в реальном времени

## Для отелей и DMC
- **Запрос услуги** — roomservice / трансфер / экскурсия прямо из приложения
- **Карта отеля** — интерактивная схема: где ресторан, spa, конференц-залы
- **Мультиязычность** — переключение EN/RU/AR одной кнопкой

## Топ-3 для этой аудитории
1. AI-консьерж — все поймут применение у себя
2. Живые опросы — вау-эффект прямо на сцене
3. Цифровая визитка + нетворкинг — практическая польза здесь и сейчас

## Чат участников
- Firebase Realtime Database — бесплатно, реальное время, идентификация по Telegram ID
